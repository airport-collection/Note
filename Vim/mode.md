# Basic Mode

There are 7 basic modes.

## Normal Mode
可以输入所有的正常的编辑器命令（除非...）。该模式也称为命令模式（command mode）。

## Visual Mode

## Command-line(Cmdline) Mode
在该模式下，可以在窗口最后一行（非内容区）输入命令。

## Ex Mode
类似于Cmdline Mode，但是输入命令后，依然在该模式下。

## Terminal-Job Mode
与终端中的作业进行交互。
