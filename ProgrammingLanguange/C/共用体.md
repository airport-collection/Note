多个成员共用一块空间，整个空间大小为最大成员所需要占用的空间。

共用体嵌套结构体的定义形式及用法：
比如对于一个整型无符号数，需要将其高16位与低16位相加，可以巧妙利用共用体存储的特点：
```c
union test
{
    struct
    {
        uint16_t i;
        uint16_t j; 
    };
    
    int a;
}

int main()
{
    union test x;
    x.a = 0x11223344;
    printf("%x\n", x.i + x.j); // 输出4466
}
```
